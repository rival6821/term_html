Google API 사용 목적

  • n8n 워크플로우 내 자동화 작업을 위한 API 호출
  • 필요 시 알림/연동용 데이터 읽기 또는 전달
  • 서비스 안정성 모니터링(비식별 로그 수준)

Google API는 사용자 요청을 처리하는 데 필요한 범위에 한해 호출하며, 요청 목적은 위에 명시한 자동화/알림 처리에 한정됩니다.

데이터 수집/보관

  • 개인식별 정보: 수집/저장하지 않음
  • 쿠키·광고 추적: 사용하지 않음
  • 서버 로그: 보안/장애 분석용 비식별 형태로 최소 보관 후 삭제

로그 데이터는 개인을 식별하거나 마케팅 목적으로 사용하지 않습니다.

OAuth / 권한

  • 현재 Google OAuth 동의 화면에서 요청하는 민감/제한된 범위는 없습니다.
  • 향후 범위가 추가될 경우, 사전 고지 및 약관/정책 업데이트 후 반영합니다.

필요 최소 범위를 원칙으로 하며, 액세스 토큰은 워크플로우 실행에 필요한 기간에 한해 관리합니다.

데이터 공유 및 제3자 제공

  • 개인정보를 저장하지 않으므로 제3자 제공 없음
  • n8n 내 연동 서비스에도 개인식별 정보 전송 없음

보안 및 사용자 권리

  • 전송 구간 보호를 위해 HTTPS 사용
  • 비식별 로그 접근 최소화
  • 문의 시 신속한 삭제·정정 안내 (수집 데이터 없음)
